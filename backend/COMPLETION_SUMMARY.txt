## 🚀 How to Use This

### Option 1: Test Locally Today (5 minutes)

```bash
cd backend

# Setup
python -m venv venv
source venv/bin/activate  # or .\venv\Scripts\Activate.ps1 on Windows
pip install -r requirements.txt

# Run server
uvicorn main:app --reload

# In another terminal: Run tests
pytest tests/ -v

# Expected: 40+ tests pass in <3 seconds ✅
```

### Option 2: Run Manual Testing (10 minutes)

Follow **TESTING.md** Phase 3-5 to manually test each endpoint with curl:

```bash
# Get test key
TEST_KEY=$(curl -s -X POST "http://localhost:8000/stripe/test-create-key?email=test@example.com&plan=all" | jq -r '.data.key')

# Test TCG endpoint
curl -H "X-API-Key: $TEST_KEY" "http://localhost:8000/tcg/price?card=Black+Lotus&game=mtg"

# Test OSHA
curl -H "X-API-Key: $TEST_KEY" "http://localhost:8000/osha/trades"

# Test Contracts
curl -H "X-API-Key: $TEST_KEY" "http://localhost:8000/contracts/categories"

# Test Onboarding
curl -H "X-API-Key: $TEST_KEY" "http://localhost:8000/onboarding/industries"
```

### Option 3: Deploy to Production (30 minutes after config)

**First: Collect 6 configuration values** (see DEPLOYMENT.md section 1)

Then deploy to Railway:
```bash
railway login
railway link
railway up
```

See **DEPLOYMENT.md** for complete step-by-step.
